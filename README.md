# 🎮 Tic Tac Toe AI

## 📌 Description

This project is a Tic Tac Toe game with an AI opponent. The AI uses an intelligent algorithm to make optimal moves, making the game challenging for the player.

## 🚀 Features

* Play Tic Tac Toe against computer 🤖
* Smart AI opponent (Minimax Algorithm)
* Detects win, loss, and draw conditions
* Simple and interactive gameplay

## 🛠️ Technologies Used

* Python
* Basic AI (Minimax Algorithm)

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/nikitapatil78927/tic_tac_toe_ai.git

2. Navigate to the folder:
   cd tic_tac_toe_ai

3. Run the program:
   python main.py

## 🎯 How It Works

* The game board is displayed in the console
* Player makes a move
* AI calculates the best possible move
* Game continues until win or draw

## 🧠 AI Logic

The AI uses the **Minimax algorithm**, which evaluates all possible moves and chooses the optimal one to maximize its chances of winning.
